## 內容哲學

**每一篇內容 = AI 藝術教學 + AI 藝術實戰小專案。**

學員讀完後，手上有一個真實可用的東西，不是「更多了解」。

錯誤寫法 → 正確寫法：
- 「認識 Midjourney 的功能」→「做出一套可以賣給客戶的品牌素材」
- 「了解 AI 圖像創作的可能性」→「完成一份可以上架 Etsy 的 AI 插畫商品」
- 「學習 Agent 的概念」→「部署一個自動生成品牌素材的 Agent 員工」

## 升級路徑

### Level 1：AI 創作力

**目標**：做出第一套用 AI 完成的品牌視覺，別人願意付錢。

- 用 Midjourney 做出第一套品牌識別視覺（Logo 風格、主視覺、色盤）
- 用 Adobe Firefly 快速完成商業授權素材，直接可交付客戶
- 用 AI 做出一套社群貼文模板，可重複使用、風格統一
- 建立個人 AI 作品集展示頁，讓客戶可以主動找上門
- 完成第一份 AI 設計提案，拿到第一個客戶的肯定

通關條件：有一套作品，別人願意付錢。

### Level 2：開課賺錢

**目標**：把 AI 設計技能包裝成可以重複銷售的服務或數位商品，有真實的付款記錄。

- 設計你的第一個 AI 藝術服務套餐（品牌識別、社群素材、電商視覺）
- 定價策略：AI 設計服務如何定價，不再用時間換錢
- 建立數位商品（AI Preset、Prompt 模板、授權圖庫）的被動收入
- 個人品牌網站：作品集 + 服務頁 + 接案系統一次到位
- 第一筆收入：從接案到自建穩定客源的完整路徑

通關條件：有真實的付款記錄。

### Level 3：升級 OPC

**目標**：用 AI Agent 讓一人藝術公司自動運作，建立不需每天親自操作的收入系統。

- 建立自動生成品牌素材的 Agent 員工（輸入需求 → 輸出成品）
- 自動接單 → 自動出圖 → 自動交稿的完整工作流
- 建立 AI 藝術授權圖庫的被動收入系統（上架一次，持續進帳）
- 客服 Agent：24 小時自動回覆諮詢、自動報價、自動篩選客戶
- OPC 完整架構：一個人的 AI 藝術公司怎麼跑

通關條件：有一個不需要每天親自操作的收入系統。

## 課程卡必要欄位

每一門課必須回答：

| 欄位 | 說明 |
|---|---|
| `outcome` | **你會做出什麼** — 具體的交付物，不是知識點 |
| `durationMinutes` | 需要多久 |
| `audience` | 適合誰 |
| `tool` | 使用的 AI 工具（Midjourney / Firefly / Stable Diffusion / ComfyUI 等）|
| `reward` | 完成獎勵 — 遊戲化成就感 |
| `missionLabel` | 任務標籤，例如「MISSION 01」 |
| `smallTask` | 今日小任務（title / description / deliverable）|
| `steps` | 步驟陣列（每步含 prompt + checkpoint）|
| `copyPrompt` | 主要 Prompt 咒語，可直接複製使用 |
| `completionChecks` | 過關條件清單 |

## 內容語氣

- 像教練對學員說話，不像教授對學生
- 標題直接說結果：「做出」「完成」「建立」「部署」
- 不說「本課程旨在...」→ 說「這一課你會做出...」
- 每一句話都指向行動
- 說「你」不說「您」
